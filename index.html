01 — The Problem

Avant / Après · Before / After

Le moteur transforme une approche réactive en une stratégie prédictive. The engine transforms a reactive approach into a predictive strategy.

Avant · Before
  • Appels clients au hasard / Random customer calls
  • Taux de succès rétention ~5% / ~5% retention success rate
  • Aucune visibilité prédictive / No predictive visibility
  • Réaction après résiliation / Reaction after cancellation
  • Données silotées, non exploitées / Siloed, unexploited data
Après · After
  • 50 clients prioritaires chaque lundi / 50 priority customers every Monday
  • Taux de succès cible 25–35% / Target success rate 25–35%
  • Score de Risque IA expliqué / Explained AI Risk Score
  • Action 90 jours avant le départ / Action 90 days before leaving
  • Pipeline serverless fiable et sobre / Reliable, frugal serverless pipeline

04 — Product Methodology

3 Piliers · 3 Pillars

La méthodologie couvre le cycle de vie complet du modèle, le contrôle humain, et la conduite du changement. The methodology covers the full model lifecycle, human control, and change management.

PILIER 01 · PILLAR 01
Cycle de Vie ML
ML Lifecycle
  • Data Quality Gate avant tout entraînement
  • Autopilot frugal (10 candidats, Spot, ~20€)
  • Batch scoring hebdo, pas d'endpoint 24/7
  • Model Monitor — alerte dérive PSI >0.2
  • Champion vs Challenger avant déploiement
  • Rollback possible en 1h (90j de rétention)

"La surveillance ne ré-entraîne jamais automatiquement. Elle alerte un humain."
Monitoring never auto-retrains. It alerts a human.

PILIER 02 · PILLAR 02
Human in the Loop
HITL
  • Agent valide avant tout contact client
  • Niveau 1 : agent hebdo (30 min)
  • Niveau 2 : DPM + Data mensuel (45 min)
  • Niveau 3 : Direction trimestriel (60 min)
  • Feedbacks → correction labels → re-training
  • Taux confirmation <50% → alerte modèle

"L'IA propose. L'humain décide."
"AI proposes. Humans decide."

PILIER 03 · PILLAR 03
Conduite du Changement
Change Management
  • Phase Alpha : 2–3 agents volontaires
  • Phase Beta : équipe marketing complète
  • Phase Prod : KPI officiel, onboarding
  • Formation peer-to-peer à chaque phase
  • Démos avec cas réels anonymisés en Sprint Review
  • Documentation Confluence dès Sprint 4

"Un modèle précis que personne n'utilise ne vaut rien."
"A precise model nobody uses is worthless."
